# 🚀 Quick Start (5 Minutes)

## Step 1: Install & Configure (3 minutes)

```bash
# Navigate to project
cd appsec_final

# Install packages
pip install -r requirements.txt

# Create .env file
copy .env.example .env

# Edit .env and add your Claude API key
# CLAUDE_API_KEY=sk-ant-your-key-here
# (Get key from https://console.anthropic.com/)
```

## Step 2: Start App (30 seconds)

```bash
python run.py
# Opens http://localhost:5000 automatically
```

## Step 3: Try It (90 seconds)

1. **Register** → Create account (any email/password)
2. **Create Project** → Upload any Python/JS ZIP file
3. **Run Scan** → Click "Run Security Scan"
4. **View Findings** → Click any issue
5. **Generate Fix** → Click "🔍 Generate Fix Preview"
6. **Review Diff** → See side-by-side code comparison
7. **Apply** → Click "✅ Apply Fix"
8. **Download** → Get fixed ZIP file
9. **Report** → Generate PDF

## ✅ Done!

That's it. You now have an AI-powered security scanner that:
- ✅ Scans projects for vulnerabilities
- ✅ Fixes them with AI (not just patterns)
- ✅ Shows you exactly what changed
- ✅ Lets you download the fixed code
- ✅ Creates professional PDF reports
